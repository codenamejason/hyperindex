### Getting Started

Install the following tools:

1. nodejs (install v18) https://nodejs.org/en
   (Recommended to use a node manager like fnm or nvm)
   2.then install pnpm:
   `npm install --global pnpm`
2. Cargo https://doc.rust-lang.org/cargo/getting-started/installation.html
   `curl https://sh.rustup.rs -sSf | sh`
3. Docker Desktop https://www.docker.com/products/docker-desktop/

Using "indexly"

1. `cargo install --path codegenerator`
2. run `indexly --help` to see available commands
3. cd into folder of your choice and run: `indexly init` to get a Gravatar template
4. then run: `indexly codegen`
